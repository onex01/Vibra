# Vibra Script (.vs) — Язык программирования Vibra OS

## Обзор

Vibra Script — простой интерпретируемый язык программирования для Vibra OS.
Предназначен для скриптов, автоматизации, игр и обучения программированию.

## Запуск скриптов

```
# Из файла
run /path/to/script.vs

# Inline код
run -e print 2 + 3

# Мульти-команда
run -e var x = 10; while x > 0 { print x; x = x - 1 }
```

## Типы данных

| Тип | Пример | Описание |
|-----|--------|----------|
| Число | `42`, `-7`, `3.14` | Целые числа (i64) |
| Строка | `"hello"`, `'world'` | Строки в кавычках |
| Nil | `null`, `nil` | Пустое значение |

## Переменные

```vs
var x = 10              # Объявление переменной
x = x + 1               # Присваивание
var name = "Vibra OS"   # Строка
var flag = true          # Булево (1 = true, 0 = false)
```

## Операторы

### Арифметические
```
+   сложение
-   вычитание
*   умножение
/   деление
%   остаток от деления
```

### Сравнения
```
==  равно
!=  не равно
<   меньше
>   больше
<=  меньше или равно
>=  больше или равно
```

### Логические
```
!   отрицание
```

## Управляющие конструкции

### If/Else
```vs
var x = 10
if x > 5 {
    print "x больше 5"
} else {
    print "x меньше или равно 5"
}
```

### While
```vs
var i = 0
while i < 10 {
    print i
    i = i + 1
}
```

### For (счётчик)
```vs
for i = 1 to 10 {
    print i
}

# Обратный отсчёт
for i = 10 to 1 {
    print i
}
```

```vs
print "Hello, World!"

```

### Input (ввод)
```vs
input name
print "Привет, " + name + "!"

```

## Звук

### Beep (PC Speaker)
```vs
beep 440           # Включить звук 440 Гц (нота A4)
beep 880 500       # 880 Гц на 500мс
beep 0             # Выключить звук
```

### Мелодия
```vs
beep 262 200       # C4
beep 294 200       # D4
beep 330 200       # E4
beep 349 200       # F4
beep 392 200       # G4
beep 440 200       # A4
beep 494 200       # B4
beep 523 400       # C5
beep 0             # Тишина
```

### Пауза
```vs
sleep 100          # Пауза 100 тиков (~1 секунда)
```

## Примеры скриптов

### Калькулятор
```vs
print "=== Калькулятор ==="
var a = 10
var b = 3
print a + " + " + b + " = " + (a + b)
print a + " - " + b + " = " + (a - b)
print a + " * " + b + " = " + (a * b)
print a + " / " + b + " = " + (a / b)
```

### Факториал
```vs
var n = 5
var result = 1
var i = 2
while i <= n {
    result = result * i
    i = i + 1
}
print "5! = " + result
```

### Генератор таблицы умножения
```vs
for i = 1 to 9 {
    var line = ""
    for j = 1 to 9 {
        line = line + (i * j) + " "
    }
    print line
}
```

### Мелодия (до-ре-ми)
```vs
var notes = 262
var n = 0
while n < 8 {
    beep notes 200
    sleep 250
    notes = notes + 30
    n = n + 1
}
beep 0
```

## Ограничения

- Максимум 16 аргументов в командной строке shell
- Нет поддержки массивов (пока)
- Нет поддержки функций (пока)
- Нет поддержки файлового ввода/вывода
- Числа — только целые (i64)
