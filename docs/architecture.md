# Архитектура Vibra OS

## Ядро

**Тип:** Модульное монолитное ядро  
**Язык:** Rust (no_std)  
**Загрузчик:** Limine (UEFI)

### Модули ядра:

1. **serial** — драйвер последовательного порта COM1 (0x3F8)
2. **memory::pmm** — физический менеджер памяти (bitmap allocator)
3. **memory::heap** — аллокатор кучи (планируется)
4. **keyboard** — драйвер PS/2 клавиатуры (в разработке)
5. **fs** — файловая система (RamFS + FAT32, планируется)
6. **shell** — командная оболочка (планируется)

## Загрузка

1. UEFI BIOS (OVMF) загружает Limine
2. Limine читает `limine.conf` и загружает `kernel.elf`
3. Limine передаёт ядру:
   - Framebuffer (адрес, размер, pitch)
   - Memory map (карта памяти)
   - HHDM offset (Higher Half Direct Map)
4. Ядро инициализирует драйверы и входит в главный цикл

## Управление памятью

- **Физическая память:** Bitmap allocator (4 КБ страницы)
- **Виртуальная память:** Планируется (page tables)
- **Куча:** Планируется (linked list allocator)

## Ввод-вывод

- **Serial:** Port-mapped I/O (0x3F8)
- **Framebuffer:** Memory-mapped I/O (через HHDM)
- **Клавиатура:** PS/2 (порт 0x60, прерывание IRQ1)

## Файловая система

### RamFS (в разработке)
- Плоская структура (без папок на старте)
- Хранение в памяти
- Быстрый доступ

### FAT32 (планируется)
- Чтение/запись на диск
- Поддержка папок
- Совместимость с Windows/Linux
